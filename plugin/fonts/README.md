# Arial Unicode MS Font Installation

## Required Font File
Download **Arial Unicode MS** font file and place it here:
- `ARIALUNI.TTF` (Arial Unicode MS)

## Where to Get Arial Unicode MS
1. **From Windows**: `C:\Windows\Fonts\ARIALUNI.TTF`
2. **From Mac**: `/Library/Fonts/Arial Unicode.ttf`
3. **Download**: Search for "Arial Unicode MS free download" (ensure licensing compliance)

## Installation Steps
1. Copy `ARIALUNI.TTF` to this directory (`/fonts/`)
2. The PDF generator will automatically use this font
3. Supports full Unicode including ₦ (Naira) and ✓ (checkmark)

## Font Metrics
Dompdf will automatically generate font metrics (.ufm files) on first use.

## Alternative: Use DejaVu Sans
If Arial Unicode is not available, the system will fall back to DejaVu Sans which also supports Unicode characters.
