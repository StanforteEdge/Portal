# Stanforte Edge Portal PWA2

This is the new PWA2 workspace for the Stanforte Edge staff portal.

It is intentionally small at first:
- new shell
- new design direction
- new pages can be added here without touching the legacy `PWA` app

Run from the repo root:

```bash
npm run dev:pwa2
```
