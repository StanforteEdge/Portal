# Notification, Email & Reminders (Core Feature)

## Overview
Centralized notification engine for the portal. Handles in-app notifications, email alerts, and reminders for all modules (Finance, HR, Projects, etc.).

## Key Capabilities
- In-app notifications (real-time or batch)
- Email alerts (transactional, workflow, reminders)
- Scheduled reminders and escalation
- Notification templates and localization
- User notification preferences

## Why Core?
- Shared by all modules for workflow and engagement
- Consistent user experience and delivery
- Easy integration with approval workflows, audits, and triggers

- /api-spec.md: Notification endpoints
- /templates/: Email and notification templates
- /integration.md: How to trigger notifications from other modules
