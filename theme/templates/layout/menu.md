# Stanforte Edge Portal - Menu Structure

## 1. Dashboard
- Overview (All)
  - Quick Stats
  - Recent Activities
  - Notifications
- My Tasks (All)
  - Active Tasks
  - Due Soon
  - Completed Tasks
- My Requests (All)
  - Financial Requests
  - Request Status
  - Request History
- My Leave (All)
  - Leave Balance
  - Leave History
  - Upcoming Leave
- My Documents (All)
  - Recent Documents
  - Shared with Me
- My Attendance (All)
  - Today's Status
  - Monthly Overview
  - Attendance History

## 2. Tasks & Projects
- Task Board (All)
  - My Tasks
  - Team Tasks (Team Lead)
  - Task Calendar
- Projects Overview (All)
  - Active Projects
  - Project Timeline
  - Project Reports
- Create/Edit (Team Lead, Project Owner)
  - New Task
  - New Project
  - Edit Project

## 3. Requests
- Leave Requests
  - Apply Leave (All)
  - Leave Calendar
  - Leave Approvals (Team Lead, HR)
- Finance Requests
  - New Request (All)
  - Request Status
  - Approvals (Finance, Team Lead)
  - Reports (Finance)
- Incident Reports
  - Report Incident (All)
  - View Reports (HR, Team Lead)
  - Follow-ups
- Staff Appraisals
  - My Appraisal (All)
  - Team Appraisals (Team Lead)
  - Department Reviews (HR)

## 4. Documents
- My Documents (All)
  - Personal Documents
  - Shared Documents
  - Upload Document
- Team Documents (All)
  - Project Documents
  - Team Resources
  - Shared Files
- Policy Documents (All)
  - HR Policies
  - Finance Policies
  - General Policies
- Strategic Documents (Management)
  - Strategic Plans
  - Reports
  - Board Documents

## 5. Profile
- Personal Info (All)
  - Basic Details
  - Contact Information
  - Emergency Contacts
- Leave Management (All)
  - Leave Balance
  - Leave History
  - Leave Calendar
- Attendance Records (All)
  - Daily Log
  - Monthly Summary
  - Attendance Reports
- Performance History (All)
  - Appraisals
  - Achievements
  - Training Records

## 6. Settings
- Account Settings (All)
  - Profile Settings
  - Notification Preferences
  - Password Change
- System Settings (Admin)
  - User Management
  - Role Management
  - System Configuration
- Department Settings (Team Lead, HR)
  - Team Configuration
  - Leave Settings
  - Attendance Rules

Note: Access levels are indicated in parentheses:
- (All): Available to all staff
- (Team Lead): Available to team leaders
- (HR): Available to HR personnel
- (Finance): Available to finance personnel
- (Admin): Available to system administrators
- (Management): Available to executive management
